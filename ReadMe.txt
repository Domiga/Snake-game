Принцип работы приложения: 
- приложение запускается в браузере URL: http://a0567437.xsph.ru/game/index.html;
- для начала работы приложения необходимо нажать одну из четырех стрелок на клавиатуре. Вверх (38), вниз (40), влево (37), вправо (39);
- При соприкосновении с препятствиями игра перезапускается, счет обнуляется;
- Для корректной работы приложения необходимо при запуске нажать сочетание клавиш ctrl + F5.
